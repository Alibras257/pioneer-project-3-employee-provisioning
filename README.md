# Pioneer DevOps - Employee Account Provisioning Tool

## Project Overview
This project automates a simple employee onboarding workflow using a Bash script. The script processes a list of new employee names from an input file, creates a directory for each employee, generates a welcome message, and records all operations in a log file.

## Project Goals
- Automate weekly employee onboarding tasks
- Practice Bash scripting with loops, functions, and file handling
- Implement operation logging for monitoring and troubleshooting
- Maintain a structured repository with documentation and evidence
